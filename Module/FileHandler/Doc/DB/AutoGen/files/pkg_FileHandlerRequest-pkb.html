   charEncoding varchar2 :=  null ,
   isGzipped integer :=  null ,
   colText pkg_FileHandlerBase.tabClob :=  null ,
   useCache boolean :=  null
) return integer

Создаёт запрос для обработчика.

Параметры

operationCodeкод операции ( см.  pkg_FileHandlerBase )
commandTextтекст OS-команды
fileFullPathпуть к имени файла( директории)
fileMaskмаска для файлов.  Использование аналогично использованию в sql-операторе like escape ‘\’
maxListCountмаксимальное количество файлов в списке
fileDestPathпуть к имени файла-назначения для операции копирования
isOverwriteфлаг перезаписи файла ( см.  <pkg_FileHandler.FileCopy )
writeModeрежим записи в существующий файл ( см.  pkg_FileHandler.UnloadTxt )
charEncodingкодировка для выгрузки файла ( по-умолчанию используется кодировка базы)
isGzippedфлаг сжатия с помощью GZIP
colTextколлекция clob для выгрузки в файл
useCacheиспользовать ли данные кэш-директории.  По-умолчанию использовать.

Примечание

  • функция выполняется в автономной транзакции для того, чтобы обработчик мог увидеть данные запроса
