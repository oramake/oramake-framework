DirJava

function DirJava(fromPath varchar2 ,
entryType number) return number

Сохраняет список элементов каталога во временной таблице tmp_file_name.  Представляет собой оберку для соответствующей Java-функции.

Параметры

fromPathпуть к каталогу
entryTypeтип возвращаемых элементов каталога ( 1 файлы, 2 каталоги)

Замечание

  • для успешного выполнения у пользователя должны быть права на чтение каталога, а также всех его элементов, на уровне Java;

Например, для получения списка элеметов каталога “c:\temp” пользователем DOCUMENT нужно выдать права

begin
  dbms_java.grant_permission (
    'DOCUMENT',                         --пользователь
    'java.io.FilePermission',
    'c:\temp',                          --каталог
    'read'
  );
  dbms_java.grant_permission (
    'DOCUMENT',                         --пользователь
    'java.io.FilePermission',
    'c:\temp\*',                        --все элементы каталога
    'read'
  );
end;
/

Выполнение команд

ExecCommandJava

function ExecCommandJava(command in varchar2 ,
output in out nocopy clob ,
error in out nocopy clob) return number

Выполняет команду ОС на сервере.  Представляет собой оберку для соответствующей Java-функции.

Параметры

commandкомандная строка для выполнения
outputвывод команды ( stdout, возврат)
errorошибки ( stderr, возврат)

Возврат

  • код завершения команды.

Замечание

  • для успешного выполнения у пользователя должны быть права на выполнение указанной ( или любой) команды, а также права на запись и чтение дескрипторов файлов на уровне Java;

Пример ( права на выполнение любой команды для пользователя DOCUMENT)

begin
  dbms_java.grant_permission(
    'DOCUMENT',
    'SYS:java.lang.RuntimePermission',
    'readFileDescriptor',
    ''
  );
  dbms_java.grant_permission(
    'DOCUMENT',
    'SYS:java.lang.RuntimePermission',
    'writeFileDescriptor',
    ''
  );
  dbms_java.grant_permission(
    'DOCUMENT',
    'SYS:java.io.FilePermission',
    '<<ALL FILES>>',
    'execute'
  );
end;
/
