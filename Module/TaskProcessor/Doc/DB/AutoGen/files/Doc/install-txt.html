Общие замечания

Параметры установки

INSTALL_VERSIONустанавливаемая версия модуля ( Last при первоначальной установке или номер при обновлении)
LOAD_OPERATORIDлогин и пароль оператора, выполняющего установку ( формат: login/password)
LOAD_USERIDосновной пользователь для установки в БД ( формат: username/password@dbName)
LOAD_USERID2пользователь, под которым создаются роли для модуля AccessOperator ( формат: username/password@dbName)

Замечания

  • на время установки модуля выполняется приостановка работы пакетных заданий модуля;

2.3.0.2

Для каждого пользователя, которому выданы права на использование модуля ( указывается в TO_USERNAME), нужно выполнить команду:

make grant INSTALL_VERSION=2.3.0.2 LOAD_USERID=???/??? TO_USERNAME=??? LOAD_OPERATORID=???/???
