title: Установка

group: Общие замечания

Параметры установки:
INSTALL_VERSION               - устанавливаемая версия модуля ( Last при
                                первоначальной установке или номер при
                                обновлении)
LOAD_OPERATORID               - логин и пароль оператора, выполняющего установку
                                ( формат: login/password)
LOAD_USERID                   - основной пользователь для установки в БД
                                ( формат: username/password@dbName)
LOAD_USERID2                  - пользователь, под которым создаются роли
                                для модуля AccessOperator
                                ( формат: username/password@dbName)

Замечания:
- на время установки модуля выполняется приостановка работы пакетных заданий
  модуля;



group: 2.5.0

Порядок установки:

- установка в основную схему

> make install INSTALL_VERSION=2.5.0 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

Замечания:
- в случае необходимости отмены установки нужно выполнить аналогичную команду,
  заменив "install" на "uninstall" (при этом необходимо использовать
  OraMakeSystem версии не ниже 2.4.0);

group: 2.3.0.2

Для каждого пользователя, которому выданы права на использование модуля
( указывается в TO_USERNAME), нужно выполнить команду:

(code)

make grant INSTALL_VERSION=2.3.0.2 LOAD_USERID=???/??? TO_USERNAME=??? LOAD_OPERATORID=???/???

(end)

group: Первоначальная установка

Порядок установки:

- установка в основную схему

(code)

make install INSTALL_VERSION=Last LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

(end)

- создание ролей для модуля AccessOperator

(code)

make install INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

(end)

Замечания:
- при первоначальной установки деактивация/активация пакетных заданий
  обработчиков не производится;



group: Выдача прав на использование модуля

Команда для выдачи прав:

(code)

make grant LOAD_USERID=???/??? TO_USERNAME=??? LOAD_OPERATORID=???/???

(end)
