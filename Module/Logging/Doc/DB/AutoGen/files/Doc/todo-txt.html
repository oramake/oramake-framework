Общие

[+] Добавить в объект lg_logger_t статические функции, возвращающие коды уровней сообщений лога, чтобы не было необходимости обращаться к пакету pkg_Logging ( коды требуются, например, при использовании функции lg_logger_t.log).

[+] Добавить дополнительные уровни логирования для более глубокой трассировки ( как минимум TRACE2).  Пример использования: вывод объемной детальной информации.

[+] Возможность логирования сообщений большой длины ( более 32K, через тип CLOB).

[+] Добавить в справочник lg_level поле message_level_code ( с уникальным ключом), в котором должны указываться возможные коды уровня сообщений ( т.е.  без “ALL” и “OFF”) для использования в таблицах логов других модулей ( например, TaskProcessor) либо, как вариант, удалить из таблицы lg_level записи с суррогатными кодами “ALL” и “OFF”, используемыми только для изменения уровня логирования.

[+] Добавить возможность выполнения логирования в рамках текущей транзакции ( полезно для предупреждений и т.д., связанных с изменением данных).

[*] Удаление лишних сообщений при установке.  Использование скрипта oms-default.

Завершение доработок

[*] После перехода в модуле Scheduler на использование поля level_name вместо level_description (в версиях после 5.1.0) убрать заполнение по умолчанию значения level_description из level_name в скрипте Install/Data/Last/lg_level.sql.

[*] Удалить устаревшую константу pkg_Logging.Warning_LevelCode( DEPRECATED) после прекращения ее использования в прикладных модулях.

[*] После прекращения использования в прикладных модулях удалить параметр packageName в функции lg_logger_t.getLogger.

create table lg_level (
   level_code varchar2(10) not null ,
   message_level_code as ( case when level_code not in ('ALL', 'OFF') then level_code end ) ,
   level_order number(3) not null ,
   level_name varchar2(100) not null ,
   level_description varchar2(255) ,
   date_ins date default sysdate not null ,
   constraint lg_level_pk primary key (level_code) using index tablespace &indexTablespace ,
   constraint lg_level_uk_message_level_code unique (message_level_code) using index tablespace &indexTablespace
)
