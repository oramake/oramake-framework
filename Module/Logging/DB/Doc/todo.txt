Title: Доработки



group: Общие

[+] Добавить в объект <lg_logger_t> статические функции, возвращающие коды
    уровней сообщений лога, чтобы не было необходимости обращаться к пакету
    <pkg_Logging> ( коды требуются, например, при использовании функции
    <lg_logger_t.log>).

[+] Добавить дополнительные уровни логирования для более глубокой трассировки
    ( как минимум TRACE2). Пример использования: вывод объемной детальной
    информации.

[+] Возможность логирования сообщений большой длины ( более 32K, через тип
    CLOB).

[+] Удобная выборка сообщений собственной сессии ( например, создать специальное
    представление).

[+] Добавить в справочник <lg_level> поле message_level_code ( с уникальным
    ключом), в котором должны указываться возможные коды уровня сообщений
    ( т.е. без "ALL" и "OFF") для использования в таблицах логов других
    модулей ( например, TaskProcessor) либо, как вариант, удалить из таблицы
    <lg_level> записи с суррогатными кодами "ALL" и "OFF", используемыми только
    для изменения уровня логирования.

[*] После прекращения использования в прикладных модулях удалить параметр
    packageName в функции <lg_logger_t.getLogger>.

[+] Добавить возможность выполнения логирования в рамках текущей транзакции
    ( полезно для предупреждений и т.д., связанных с изменением данных).

[*] Удаление лишних сообщений при установке. Использование скрипта oms-default.

[*] Изменить тип колонки date_ins <lg_log> на "timestamp with time zone".



group: Завершение доработок

    вместо временного решения по заполнению parent_log_id из предыдущей
    добавленной записи, которое может работать некорректно
    ( удалить <lg_log_bi_define>, <pkg_LoggingInternal.beforeInsertLogRow>).

[*] Удалить устаревшую константу <pkg_Logging.Warning_LevelCode( DEPRECATED)>
    после прекращения ее использования в прикладных модулях.

[*] Валидировать ограничения NOT NULL по полям sessionid, level_code, log_time
    таблицы <lg_log> (в какой-то последующей после 2.0.0 версии).
