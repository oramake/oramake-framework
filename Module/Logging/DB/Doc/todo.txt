group: Общие

[+] Добавлить в функцию <lg_logger_t.mergeContextType> параметр autoDeleteFlag
    для возможности создания "локальных временных" контекстов выполнения
    (например, уровня определенного скрипта модуля). Такие контексты должны
    автоматически удаляться из таблицы <lg_context_type> при отсутствии ссылок
    на них в логе в рамках процесса очистки записей лога.

[+] Добавить дополнительные уровни логирования для более глубокой трассировки
    ( как минимум TRACE2). Пример использования: вывод объемной детальной
    информации.

[+] Возможность логирования сообщений большой длины ( более 32K, через тип
    CLOB).

[+] Добавить возможность выполнения логирования в рамках текущей транзакции
    ( полезно для предупреждений и т.д., связанных с изменением данных).

[*] Удаление лишних сообщений при установке. Использование скрипта oms-default.
