convertNameInCase

function convertNameInCase(nameText varchar2 ,
formatString varchar2 ,
caseCode varchar2 ,
sexCode varchar2 default null) return varchar2

Функция преобразования ФИО к указанному падежу.  Порядок слов в формате и в переданной строке должен совпадать.  Двойные фамилии должны отделяться друг от друга знаком “-”, при этом количество пробелов до и после знака не важно.

Входные параметры

nameTextСтрока для преобразования
formatStringФормат строки для преобразования
caseCodeКод падежа преобразования
sexCodeПол

Возврат

строка в указанном падеже.
