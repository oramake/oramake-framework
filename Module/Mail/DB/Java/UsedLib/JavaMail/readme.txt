title: JavaMail

Для работы с почтой по протоколам SMTP, POP3, IMAP используется библиотека
JavaMail ( источник <http://java.sun.com/products/javamail/>).

Версия:
- javamail 1.3.1

Документация:
- readme ( <http:javamail-1.3.1/README.txt>);
- API documentation ( <http:javamail-1.3.1/docs/javadocs/index.html>);
- API Design Specification ( <http:javamail-1.3.1/docs/JavaMail-1.2.pdf>);
- Guide for Service Providers ( <http:javamail-1.3.1/docs/Providers.pdf>);

Установка:
- необходимо наличие библиотеки <JAF>;
- для подключения к почтовому сереверу необходимы дополнительные права на
  уровне Java, которые выдаются пользователю при установке скриптом
  <Install/Grant/javamail.sql>;
