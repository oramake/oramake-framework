Title: Доработки

[*] В поле password таблицы <ml_fetch_request> следует сохранять пароль в
    зашифрованном виде ( с помощью функций пакета pkg_OptionCrypto модуля
    Option).

[+] Возможно ( сравнительно несложно) реализовать отправку сообщений с
    вложенными сообщениями.

[!] Следует пересмотреть обязательность колонки file_name в ml_attachment.

[+] Возможно реализовать получение html-сообщений.

[+] Добавить наряду с addAttachment и addHTMLImageAttachment
    одноимённые процедуры.

[*] При необходимости, следует подумать над сохранением обрезаемых
    параметров письма.

[!] Функция pkg_Mail.sendMessage должна возвращать количество попыток
    обработки сообщений, а не количество отправленных сообщений.

[*] Возможно следует для подобных сообщений добавить
    новый статус "Ошибка отправки", вместо "Отправка отменена".
