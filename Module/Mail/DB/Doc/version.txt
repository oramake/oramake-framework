group: 1.7.0 ( 2010-04-22)

[+] При ошибках некорректных адресов протокола smtp
( Invalid addresses, маска <Mail::MAILBOX_INCORRECT_ERROR_MASK>), отправка сообщения отменяется
обработчиком после получения ошибки

[*] Возможность обработки нескольких smtp-серверов для одного обработчика.
Замена параметра smtpServer на smtpServerList для процедур
<pkg_MailHandler.SendMessage>, <pkg_MailHandler.NotifyError>

[*] Вместо одного заданного альтернативного smtp сервера ( <pkg_Mail.GetAlternativeSmtpServer>)
используется два ( <pkg_Mail.GetMassDistributionSmtpServer>, <pkg_Mail.GetFaxSenderSmtpServer>)
