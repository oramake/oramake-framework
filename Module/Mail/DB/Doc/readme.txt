title: Описание

Модуль предназначен для отправки и получения сообщений электронной почты.

Возможности:
- получение и сохранение в БД почтовых сообщений ( поддерживаются вложения и
  вложенные сообщения);
- обработка полученных и отправка ответных сообщений ( с вложениями) в рамках
  транзакции;
- непосредственная отправка письма с вложением ( без сохранения в БД);
- получение почты по протоколам POP3 и IMAP, отправка - по SMTP ( возможно
  использование разных SMTP-серверов);

Дополнительные возможности:
- поддержка логических цепочек сообщений ( например, запрос-ответ);
- автоматическое удаление сообщений по истечению срока жизни;
- нотификация в случае ошибок при оправке сообщений;
- отправка сообщений в виде HTML;

Использование:
- в простейшем случае обработку почтовых сообщений можно выполнить следующим
  образом

  (start code)

  loop
    messageId := pkg_Mail.getMessage( url);
    exit when messageId is null;

    --обрабатываем полученное сообщение

  end loop;

  commit;

  (end)

  при этом в случае отсутствия почтовых сообщений для обрработки, функция
  <pkg_Mail.getMessage> будет вызывать <pkg_Mail.fetchMessage> для получения
  писем из почтового ящика.

- для явного контроля над подключением к почтовому ящику можно использовать
  цикл:

  (start code)

  loop
    nFetch := pkg_Mail.fetchMessage( url);
    exit when nFetch = 0;

    loop
      messageId := pkg_Mail.getMessage( recipientAddress => recipientAddress);
      exit when messageId is null;

      --обрабатываем полученное сообщение

    end loop;

    commit;

  end loop;

  (end)
