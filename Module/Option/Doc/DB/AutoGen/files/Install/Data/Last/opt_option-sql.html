Имя промышленной БД, к которой относится установка ( при установке в тестовую БД должно указываться имя соответствующей промышленной БД), по умолчанию определяется по LOAD_USERID.
LocalRoleSuffix_OptionSName constant varchar2(50) := 'LocalRoleSuffix'
Короткое наименование параметра “Суффикс для ролей, с помощью которых выдаются права на все параметры, созданные в локально установленном модуле Option”.
Определяет суффиксы для ролей, с помощью которых выдаются права на все параметры, созданные в определенной промышленной БД ( настройки могут модифицироваться в пользовательской версии).
