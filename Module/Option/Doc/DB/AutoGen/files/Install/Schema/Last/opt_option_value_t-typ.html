opt_option_value_t

Настроечный параметр с текущим используемым значением.  Набор отличается от полей представления v_opt_option_value добавлением поля encrypted_string_value ( при этом значение в поле string_value всегда указывается в незашифрованном виде).

SVN root: Oracle/Module/Option
