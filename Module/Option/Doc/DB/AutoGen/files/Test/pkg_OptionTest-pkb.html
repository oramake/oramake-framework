Замечания

  • для проверки отсутствия параметра нужно указать optionId равным null;
  • для проверки отсутствия значения параметра нужно указать соответствующий Id значения ( valueId, testValueId, usedValueId) равным null;
  • по умолчаю логически удаленные записи не учитываются, для проверки логического удаления нужно указать в соответствующих параметрах ( optionDeleted, valueDeleted, testDeleted) значение 1;
