title: Версии



group: 2.7.4 ( 2015-01-30)
(<Установка::2.7.4>)

[!] Исправлена орфографическая ошибка в реализации <pkg_Common::numberToWord> в
    слове "миллиард".



group: 2.7.3 ( 2014-05-20)

[!] В функции для склонения ФИО по падежам в случае,
    если не указан пол и в исходной строке нет отчества
    пол берется женский.



group: 2.7.2 ( 2014-02-25)

[*] Функция для склонения ФИО по падежам приведена к
    соответствию корпоративным стандартам разработки:
    - изменено название функции с pkg_Common.convertNameOnCase
      на <pkg_Common.convertNameInCase>
    - убраны сокращения в названиях параметров функции (
      "formatStr" -> "formatString")
    - названия констант с кодом падежей изменены с
      caseTypeCode на caseCode
    - названия констант с кодом пола изменены с
      sexTypeCode на sexCode и в качестве значений
      использованы значения констант



group: 2.7.0 ( 2013-12-24)

[*] Добавлена функция для склонения ФИО по падежам.



group: 2.6.0 ( 2012-10-18)

[*] Письма с нотификацией из тестовых БД по умолчанию стали отправляться на
    отдельный почтовый адрес.

[*] Уточнено документирование и форматирование функций пакета <pkg_Common>,
    добавлены комментарии к таблицам модуля.

[*] OMS-файлы модуля обновлены до версии 1.6.5.



group: 2.5.13 ( 2011-12-28)

[+] Добавлена перегруженная функция <pkg_Common.split( CLOB)>, предназначенная
    для работы со строкой типа CLOB.

[*] oms-файлы модуля обновлены до версии 1.6.1



group: 2.5.12 ( 2011-10-07)

[+] В справочник <cmn_database_config> добавлено поле ip_address_production,
    содержащее IP адрес промышленного сервера БД

[+] Добавлена функция <pkg_Common.getIpAddress>, возвращающая IP адрес
    текущего сервера БД

[*] Изменена функция <pkg_Common.isProduction>. Теперь если
    поле ip_address_production в таблице <cmn_database_config> не пустое,
    то статус isProduction (значение 1) возвращается только при
    совпадении значения поля ip_address_production с IP адресом
    текущего сервера БД

[*] oms-файлы модуля обновлены до версии 1.5.4
