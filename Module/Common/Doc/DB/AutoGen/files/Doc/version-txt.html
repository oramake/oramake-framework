[+] Добавлена возможность выдачи прав на использование модуля отдельному пользователю ( ранее права выдавались только для всех пользователей).

2.8.0 ( 2016-02-08)

[-] Из модуля исключен функционал автоматического склонения ФИО по падежам в связи с переносом его в отдельный модуль ConvertNameInCase ( для исключения зависимости модуля Common от модуля AccessOperator).

[*] Определение имени экземпляра в функции pkg_Common.getInstanceName стало выполняться с помощью sys_context вместо dbms_utility.get_parameter_value ( т.к.  этот способ не требует прав на v_$parameter).

[*] Определение SID текущей сессии стало выполняться с помощью sys_context вместо выборки из v$mystat ( т.к.  этот способ не требует дополнительных прав).

[*] Добавлен скрипт для удаления модуля из БД.

2.7.5 ( 2016-02-11)

[!]  Исправлена функция разделения объекта типа Clob по заданному разделителю pkg_Common.split( CLOB).

2.7.4 ( 2015-01-30)

(<Установка::2.7.4>)

[!]  Исправлена орфографическая ошибка в реализации pkg_Common::numberToWord в слове “миллиард”.

2.7.3 ( 2014-05-20)

[!]  В функции для склонения ФИО по падежам в случае, если не указан пол и в исходной строке нет отчества пол берется женский.

2.7.2 ( 2014-02-25)

[*] Функция для склонения ФИО по падежам приведена к соответствию корпоративным стандартам разработки:

  • изменено название функции с pkg_Common.convertNameOnCase на <pkg_Common.convertNameInCase>
  • убраны сокращения в названиях параметров функции ( “formatStr” -> “formatString”)
  • названия констант с кодом падежей изменены с caseTypeCode на caseCode
  • названия констант с кодом пола изменены с sexTypeCode на sexCode и в качестве значений использованы значения констант

[*] Добавлена функция для склонения ФИО по падежам.

[*] oms-файлы модуля обновлены до версии 1.5.4
