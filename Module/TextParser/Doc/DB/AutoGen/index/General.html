[+] Добавлен пакет pkg_TextParserTest.

(Установка::1.4.0)
[+] При разборе данных в формате CSV с помощью tpr_csv_iterator_t добавлена возможность задания символа-разделителя полей, что позволяет, например, разбирать файлы, сохраненные из MS Excel как “Текстовые файлы ( с разделителями табуляции)”.
(Установка::1.2.1)

[+] При получении числовых полей из значений ( тип tpr_csv_iterator_t) предварительно удаляются пробелы для исключения ошибки “character to number conversion error”.
