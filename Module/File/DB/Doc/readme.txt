title: Описание

Модуль обеспечивает работу с файлами и внешними программами из Oracle.

Основные возможности:
- файловые операции;
- загрузка данных;
- выгрузка данных;
- выполнение команд на сервере;

Файловые операции и загрузка/выгрузка данных работают для файлов:
- локальных ( на сервере, на котором работает БД);

  Пример: "c:\temp\tmp.txt"

- расположенных на Windows-шаре;

  Пример: "\\test\tmp\tmp.txt"

- доступных по FTP;

  Пример: "ftp://testUser:TestPasswd@ftp.server.com//home/transfer/Test/In1"

- доступных по SFTP;

  Пример: "sftp://testUser:TestPasswd@ftp.server.com/tmp.txt"


- доступных по HTTP ( только проверка наличия, копирования из, загрузка данных
  в CLOB или BLOB);

  Пример: "http://ya.ru"
