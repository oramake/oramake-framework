Title: Ошибки



group: Общие

[!] Не закрываются курсоры, открытые через Java Source.

[!] При попытке создать пустой файл с помощью процедуры
    <pkg_FileOrigin.unloadClobToFile> возникает ошибка
    "ORA-29532: Java call terminated by uncaught Java exception: java.lang.IllegalArgumentException: CLOB object is null".



group: Особенности работы

[*] При выгрузке данных в файл с помощью процедур
    <pkg_FileOrigin.unloadClobToFile> ( без указания кодировки файла) и
    <pkg_FileOrigin.unloadTxt> вместо символа с кодом 0x98 выгружается символ
    с кодом 0x3F, т.к. символа с кодом 0x98 нет в кодировке Windows-1251.
    Для решения проблемы следует для выгрузки бинарных данных ( либо текстовых
    данных в другой кодировке) использовать процедуру
    <pkg_FileOrigin.unloadBlobToFile>.
