title: edtFTPj

Для работы с файлами по протоколам FTP и SFTP используется библиотека edtFTPj
( источник <http://www.enterprisedt.com>).

Версия:
- edtFTPj/Pro 1.5.7

Документация:
- <http:edtftpj-pro-1.5.7/readme.html>

Установка:
- для подключения по SFTP необходимы дополнительные права на уровне Java,
  которые выдаются пользователю при установке скриптом
  <Install/Grant/Last/grant-edtFTPj.sql>;

Лицензия:
- edtFTPj/PRO Enterprise Developer License + 1 Year Updates/Support
  ( license.jar в текущем каталоге);
