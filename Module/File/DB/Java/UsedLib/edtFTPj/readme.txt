title: edtFTPj

Для работы с файлами по протоколу FTP используется библиотека edtFTPj
( источник <http://www.enterprisedt.com>).

Версия:
- edtFTPj/Free 2.4.0

Документация:
- <http:edtftpj-2.4.0/readme.html>

Установка:
- для подключения по FTP необходимы дополнительные права на уровне Java,
  которые выдаются пользователю при установке скриптом
  <Install/Grant/Last/grant-edtFTPj.sql>;
