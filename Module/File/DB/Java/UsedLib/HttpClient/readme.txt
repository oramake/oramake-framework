title: HttpClient

Для работы с файлами по протоколу HTTP используется библиотека HttpClient
проекта Apache HttpComponents ( источник <http://hc.apache.org>).

Версия:
- HttpClient 4.3.6 ( <http://hc.apache.org/httpcomponents-client-ga/index.html>)

Версии дополнительных библиотек:
- Apache Commons Codec 1.6
  ( <http://commons.apache.org/codec>)
- Apache Commons Logging 1.1.3
  ( <http://commons.apache.org/proper/commons-logging>)
- HttpCore 4.3.3
  ( <http://hc.apache.org/httpcomponents-core-4.3.x/index.html>)
- HC Fluent 4.3.6
  ( <http://hc.apache.org/httpcomponents-client-4.3.x/fluent-hc/project-reports.html>)

Документация:
- Tutorial <http:httpcomponents-client-4.3.6/tutorial/html/index.html>
- Настройки логирования ( используется java.util.logging)
  <http://hc.apache.org/httpcomponents-client-4.3.x/logging.html>
- HttpClient Source Xref ( with JavaDoc)
  <http://hc.apache.org/httpcomponents-client-ga/httpclient/xref/index.html>
- HttpCore Source Xref ( with JavaDoc)
  <http://hc.apache.org/httpcomponents-core-4.3.x/httpcore/xref/index.html>
- HC Fluent Source Xref ( with JavaDoc)
  <http://hc.apache.org/httpcomponents-client-ga/fluent-hc/xref/index.html>

Установка:
- для обращения по HTTP необходимы дополнительные права на уровне Java,
  которые выдаются пользователю при установке скриптом
  <Install/Grant/Last/grant-HttpClient.sql>;
