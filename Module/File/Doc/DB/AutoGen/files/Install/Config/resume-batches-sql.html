Install/Config/resume-batches.sql

Восстанавливает выполнение заданий в БД.

Выполняемые действия

  • разрешает запуск новых заданий через dbms_job, устанавливая значение параметра JOB_QUEUE_PROCESSES равным ранее сохраненному значению;
  • удаляет представление, использовавшееся для сохранения значения параметра JOB_QUEUE_PROCESSES;

Параметры

resumedViewNameимя представления со значением параметра JOB_QUEUE_PROCESSES для восстановления

Макропеременные

forcedJobQueueProcessesустанавливаемое значение параметра JOB_QUEUE_PROCESSES ( по умолчанию берется ранее сохраненное в представлении)
