[+] Добавить процедуру удаления файлов из каталога по маске ( списку масок) в связи с частым использованием данного функционала в прикладных модулях ( особенно в автотестах).

[+] Реализовать функцию FtpFile.renameTo с помощью ftp-команды “rename” для обеспечения эффективного выполнения процедуры pkg_FileOrigin.fileMove для ftp-файлов на одном сервере.

[+] Реализовать отдельную функцию, возвращающую размер файла ( при этом она может быть реализована для HTTP на основе значения заголовка “Content-Length” не смотря на невозможность получения списка файлов каталога).
