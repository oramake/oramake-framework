Product Overview



edtFTPj/PRO offers a set of robust and powerful, yet easy-to-use classes that make transferring files using the FTP, FTPS and SFTP protocols a breeze. 
edtFTPj/PRO is the Professional version of the very popular, free, open-source edtFTPj - the leading Java FTP client.  Its open-source heritage and its active user community makes its FTP engine one of the most widely tested and robust such products available for Java.

edtFTPj/PRO contains all the features of edtFTPj plus a range of usability and security features that assist the professional software developer to achieve more secure FTP functionality more quickly.  These features include:

·     Secure File Transfer through the FTPS (or FTP-over-SSL) protocol.
·     Secure File Transfer through the SFTP (or SSH File Transfer Protocol) protocol.
·     Multiple File Transfers with customizable file-filtering.
·     Single-method Directory Transfers.
·     Support for Proxies.
