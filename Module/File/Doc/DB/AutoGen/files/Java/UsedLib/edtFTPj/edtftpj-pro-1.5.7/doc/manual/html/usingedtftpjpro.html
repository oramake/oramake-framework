Key Classes

The API Documentation presents in-depth programmer's information on each class of the library.  For more task-oriented documentation please refer to the How to section.  Advice on obtaining technical support is given in Getting Help.

edtFTPj/PRO consists primarily of classes that implement the specific file transfer protocols. In addition, a factory class that produces client instances based on protocol is provided. Classes are described below:

Class
Description
FTPClientFactory
Used to create client instances based on the provided protocol.
FTPClient
Supports "plain" FTP, i.e. RFC 959 FTP
SSLFTPClient
Supports FTPS (sometimes known as FTP over SSL)
SSHFTPClient
Supports SFTP (secure transfers using SSH)
ProFTPClient
Supports "plain" FTP. Supports directory transfers and IPV6.
