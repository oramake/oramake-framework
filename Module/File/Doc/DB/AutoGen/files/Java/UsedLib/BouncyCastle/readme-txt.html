BouncyCastle

Библиотека требуется для возможности работы по SFTP через edtFTPj из Oracle 9i.

Библиотека представляет собой криптопровайдера, предоставляющего, в частности, отсутствующий в Oracle 9i алгоритм шифрования RSA, необходимый для установки SFTP-подключения ( в Oracle 10g этот алгоритм доступен по умолчанию).

Версия

  • 1.3.9

Документация

  • http:jce-jdk12-139/index.html

Установка

  • для установки и использования необходимы дополнительные права на уровне Java, которые выдаются пользователю при установке скриптом <Install/Grant/Last/grant-BouncyCastle.sql>;
