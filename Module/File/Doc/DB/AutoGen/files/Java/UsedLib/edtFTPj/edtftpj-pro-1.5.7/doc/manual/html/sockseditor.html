SOCKS Editor
   
edtFTPnet/PRO supports SOCKS proxies.  ExFTPConnection's SOCKS proxy configuration is accessible through the SocksProxySettings property.  A visual design tool is available at design-time by pressing the ellipsis-button in the SocksProxySettings property-value field.
