FTPS - Securing FTP with TLS

RFC4217, "Securing FTP with TLS" by Ford-Hutchinson, et al, is a specification for realizing RFC2228, "FTP Security Extension" using TLS.  RFC2228 is a generic specification which is designed to be applicable to a wide range of security protocols.  Ford-Hutchinson uses appropriate elements of the RFC2228 to secure FTP via TLS.  This form of Secure FTP is called FTPS.  While the spirit of RFC2228 is certainly followed, certain liberties are taken in order to better conform to TLS practices.
