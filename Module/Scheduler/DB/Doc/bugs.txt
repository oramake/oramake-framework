Title: Ошибки

[!] Результат вычисления даты следующего запуска батча на основе дней недели
    ( <pkg_Scheduler.setNextDate>) зависит от настроек NLS_TERRITORY сессии
    батча, т.к. в пакете pkg_Scheduler используется функция
    "to_char( :date, 'd')" ( например, при NLS_TERRITORY='AMERICA' для
    понедельника функция вернет 2 вместо 1).

[!] При установке в задании jobResultId равным Skip_ResultId, возникает ошибка
    при записи в лог сообщения о завершении задания, если в задании не было
    явно установлено значение jobResultMessagе:
(code)

ORA-01400: cannot insert NULL into ("<DbName>"."SCH_LOG"."MESSAGE_TEXT")

(end)
    Необходимо добавить установку текста сообщения по умолчанию для
    Skip_ResultId ( во внутренней процедуре ExecJob процедуры ExecBatch).
