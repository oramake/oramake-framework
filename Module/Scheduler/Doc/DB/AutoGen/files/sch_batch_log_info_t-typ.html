sch_batch_log_info_t

Информация из лога выполнения пакетного задания.  Тип используется в представлении v_sch_batch для исключения неэффективного выполнения запроса.  Проблема выражалась в длительном выполнении функции pkg_Scheduler.findBatch в случае вызова с минимальным набором заданных параметров ( например, rowCount и operatorId).
