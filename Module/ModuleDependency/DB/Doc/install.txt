title: Установка

Реализуется с помощью целей сборки модуля из make-целей <Установка модуля в БД>.
Для корректной установки должна быть установлена актуальная версия OMS
( см. <Версия OMS-файлов>).

Параметры установки:
INSTALL_VERSION               - устанавливаемая версия модуля ( Last при
                                первоначальной установке или номер при
                                обновлении)
LOAD_OPERATORID               - логин и пароль оператора, выполняющего установку
                                ( формат: login/password)
LOAD_USERID                   - основной пользователь для установки в БД
                                ( формат: username/password@dbName)
LOAD_USERID2                  - пользователь для установки в БД 
                                батчей выгрузки зависимостей all_dependencies
                                ( формат: username/password@dbName)
                                

group: Первоначальная установка

Предварительные требования:

- Выдать права на таблицу ss_file модуля Exchange/Module/SvnSearch

Порядок первоначальной установки:

- установка объектов модуля

(code)

$ make install INSTALL_VERSION=Last LOAD_USERID=exchange/???@ExchangeP LOAD_OPERATORID=???/???

(end)

- выдача прав на выгрузки системной информации о зависоимостях объектов 
  для исследуемых БД. Выдача осуществляется для пользователей dbLink.

(code)

$ make grant INSTALL_VERSION=Last LOAD_USERID=exchange/???@ExchangeP LOAD_OPERATORID=???/??? TO_USERNAME=???

(end)

- установка батчей для выгрузки системной информации о зависоимостях объектов 
  для исследуемых БД.

(code)

$ make install INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

(end)



group: Удаление модуля из БД

- Удаление из основной схемы

(code)

$ make uninstall INSTALL_VERSION=Last LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

(end)

- Удаление из схем, выгружающих системную информацию о зависоимостях объектов 
  для исследуемых БД.

(code)

$ make uninstall INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

(end)
