getTestOperatorId

function getTestOperatorId(
   baseName varchar2 :=  null ,
   login varchar2 :=  null ,
   roleSNameList cmn_string_table_t :=  null
) return integer

Возвращает Id тестового оператора.  Если тестового оператора не существует, он создается, если существует, то выданные ему роли корректируются согласно списку ( если он указан).

Параметры

loginЛогин оператора ( при задании используется в качестве пароля)
baseNameУникальное базовое имя оператора ( используется для формирования логина, по которому затем проверяется наличие оператора).  Может быть задан либо login либо baseName.
roleSNameListСписок кратких наименований ролей, которые должны быть выданы оператору ( по умолчанию роли не проверяются)

Возврат

Id оператора
