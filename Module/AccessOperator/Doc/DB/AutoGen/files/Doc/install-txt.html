Параметры установки

INSTALL_VERSIONустанавливаемая версия модуля ( Last при первоначальной установке или номер при обновлении)
LOAD_OPERATORIDлогин и пароль оператора, выполняющего установку ( формат: login/password)
LOAD_USERIDосновной пользователь для установки в БД ( формат: username/password@dbName)

Установка тестового пакета

make install-test LOAD_USERID=???/??? LOAD_OPERATORID=???/???

Выдача прав на использование модуля

  • выдача прав на использование тестовых утилит AccessOperator выполняется с помощью скрипта Test/Grant/run.sql с указанием в качестве первого параметра пользователя, которому выдаются права;
