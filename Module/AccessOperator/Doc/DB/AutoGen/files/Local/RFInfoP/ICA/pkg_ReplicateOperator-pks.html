copyOperator

function copyOperator(targetDbLink varchar2 default null ,
isForce integer default null ,
toBlockNumber integer default null ,
targetOperatorId integer default null) return integer

Функция копирования данных по операторам и возвращает число внесенных изменений.

Входные параметры

targetDbLinkлинк с БД назначения ( иначе данные копируются во все требуемые БД)
isForceесли 1, то выполняется принудительное ( полное) обновление вне зависимости от наличия событий изменения данных
toBlockNumberномер блока изменений, до которого выполняется обновление ( включительно, по умолчанию все изменения, игноруется если установлен isForce)
targetOperatorIdИД реплицируемого оператора

Возврат

recordNumberКоличество добавленных записей

Замечание

  • данные копируются только по тем линкам, которые прописаны в таблице rp_action с типом действия CopyOperator_ActionTypeCode.

( body::copyOperator).

Выполняет копирование данных по операторам и возвращает число внесенных изменений.

Входные параметры

targetDbLinkлинк к БД назначения
minBlockNumberминимальный номер блока событий
maxBlockNumberмаксимальный номер блока событий
isForceполное обновление ( без учета наличия событий изменения данных), если передано значение 1
asStaffFlagФлаг добавления постфикса “_as_staff” к логину оператора
targetOperatorIdИД реплицируемого оператора

Возврат

recordNumberКоличество добавленных записей

Замечения

  • в случае, если производится полное обновление данных ( isForce = 1) значение параметров minBlockNumber и maxBlockNumber игнорируется

( body::processCopyOperator).
