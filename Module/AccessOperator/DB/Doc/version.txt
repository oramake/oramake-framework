(<Установка::3.13.0>)

(<Установка::3.12.1>)

    том числе с заменой поля short_name на role_short_name).



group: 3.12.0
(<Установка::3.12.0>)

[+] Добавлена отдельная роль для изменения паролей
    операторов, над которыми уже производились
    какие-либо действия(изменение персональных данных).
    С помощью этой роли в апп части будет блокироваться
    соответствующая вкладка с изменением пароля.
