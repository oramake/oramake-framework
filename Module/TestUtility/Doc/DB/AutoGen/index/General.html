pkg_Tests.pkbpkg_TestUtility.pkb
Модуль TestUtility содержит набор функций для создания тестовых сценариев по внутреннему тестированию различных модулей.
[!]  В функции pkg_TestUtility.compareRowCount( func, cursor) возникает ошибка в случае наличия в курсоре имен колонок, отличающихся префиксом ( например, первая goods_link_flag, следующая descendant_goods_link_flag).
Установка выполняется в целевую БД.
Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.
