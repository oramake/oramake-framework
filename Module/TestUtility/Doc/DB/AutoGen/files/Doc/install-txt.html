Общие замечания

Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.  Для корректной установки должна быть установлена актуальная версия OMS ( см.  Версия OMS-файлов).

Параметры установки

INSTALL_VERSIONустанавливаемая версия модуля ( Last при первоначальной установке или номер при обновлении)
LOAD_OPERATORIDлогин и пароль оператора, выполняющего установку ( формат: login/password)
LOAD_USERIDосновной пользователь для установки в БД ( формат: username/password@dbName)

2.0.0

(Версии::2.0.0 ( 2021-03-18))

Порядок установки

  • установка в основную схему;
make install INSTALL_VERSION=2.0.0 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

Первоначальная установка

Установка выполняется в целевую БД.

Порядок установки

$ make install INSTALL_VERSION=Last LOAD_USERID=???/???@<DbTns> LOAD_OPERATORID=???/???

Выдача прав на использование модуля

Команда для выдачи прав

$ make grant LOAD_USERID=???/??? LOAD_OPERATORID=???/??? TO_USERNAME=???

Замечания

  • для выдачи прав всем пользователям в TO_USERNAME нужно указать public ( “ TO_USERNAME=public”), при этом у пользователя, в схему которого установлен модуль, должна быть системная привилегия “CREATE PUBLIC SYNONYM”;

Удаление модуля из БД

Команда для удаления модуля из БД

$ make uninstall INSTALL_VERSION=Last LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???
