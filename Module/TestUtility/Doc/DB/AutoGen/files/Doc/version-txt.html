1.0.4 ( 2015-12-15 )

[+] добавлен пакет pkg_Tests для стандартных тестов.  Первоначальная версия включает тест по поиску использования колонок из первичного ключа в триггера на update для указанной таблицы

1.0.3 ( 2014-08-28)

[+] Для определения состояния теста добавлена функция pkg_TestUtility.isTestFailed ( может использоваться для пропуска проверки оставшихся тестовых случаев после фиксации ошибки в тесте).

[*] При сравнении строк указывается минимальная длина, до которой они совпадают, и выводится начальные отрезки несовпадения.
