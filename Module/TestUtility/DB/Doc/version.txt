title: Версии



group: 1.1.0 ( 2018-10-09)

[+] Добавлены функции compareQueryResult для сравнения данных курсора (выборки
    из таблицы) с ожидаемыми данными в формате CSV.



group: 1.0.4 ( 2015-12-15 )

[+] добавлен пакет pkg_Tests для стандартных тестов. Первоначальная версия включает
    тест по поиску использования колонок из первичного ключа в триггера на update
    для указанной таблицы


group: 1.0.3 ( 2014-08-28)

[+] Для определения состояния теста добавлена функция
    <pkg_TestUtility.isTestFailed> ( может использоваться для пропуска проверки
    оставшихся тестовых случаев после фиксации ошибки в тесте).

[*] При сравнении строк указывается минимальная длина, до которой они
    совпадают, и выводится начальные отрезки несовпадения.


group: 1.0.2 ( 2013-04-03 )
(<Установка::Первоначальная установка>)

[*] пакет <pkg_TestUtility> устанавливается в основную схему в режиме
    authid current_user с созданием на него публичного синонима



group: 1.0.1

[+] Добавлены функции <pkg_TestUtility.addTestInfo>,
    <pkg_TestUtility.getTestTimeSecond>;



group: 1.0.0

[+] Первоначальная версия модуля.
