Do/stop.sql

Посылает команду остановки сессиям обработчиков.

Параметры

taskPatternшаблон имени задачи ( строка “<module_name>:<process_full_name>” сравнивается по like с этим шаблоном, по умолчанию без ограничений)

Замечания

  • ошибки при отправке команды сессиям игнорируются ( попытка отправки команды производится всем подходящим сессиям независимо от ошибок отправки);
