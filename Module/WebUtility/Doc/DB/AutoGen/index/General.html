[+] HTTP authentication functions.
Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.
Модуль с общими функциями для отправки HTTP запросов или запросов к web-сервисам.
[+] Implemented functions to execute HTTP requests with GET/POST methods.
Команда для установки обновления:
[+] Added constants for UTF-8 and Windows-1251 charsets.
[!]  Close HTTP request and response on error to prevent exception “ORA-29270: too many open HTTP requests”.
[+] Added wrapper functions pkg_WebUtility.getResponseXml(CHECK) and pkg_WebUtility.execSoapRequest.
[!]  Close HTTP request and response on error in Login procedure to prevent exception “ORA-29270: too many open HTTP requests”.
[!]  Fixed bug with sending request parameters (parameters with NULL value are ignored, long value sending incorrectly).
[+] Support Basic and Digest authentication.
Нужно повторно выдать права на использование модуля пользователям, которым они были выданы ранее (согласно Выдача прав на использование модуля).
[!]  Grant rights to object types when issuing rights to use module.
[+] Support download binary file from web server.
