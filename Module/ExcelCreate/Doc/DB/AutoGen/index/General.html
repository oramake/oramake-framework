[+] предоставить возможность добавлять формулы в ячейки и генерить строку итогов на их основе
Модуль предназначен для формирования документов в формате xml, который читается Excel’ем
Версия устанавливается в любую целевую БД.
Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.
