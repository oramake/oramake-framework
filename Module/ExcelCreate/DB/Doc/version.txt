title: Версии



group: 1.0.9 ( 2014-05-20 )
(<Установка::1.0.9>)

[*] для текстовых ячеек появилась возможность выборочного форматирования части
    ячейки с помощью HTML-тэгов (см. <pkg_ExcelCreate.addCell( varchar2 )>)

[+] добавлена возможность заливки фона ячейки указанным цветом
    (см. <pkg_ExcelCreate.addStyle>)
    
[+] добавлена возможность указать свой стиль для заголовочной строки в таблице
    (см. <pkg_ExcelCreate.addHeaderRow>)

[+] добавлена возможность создавать колонки без указания их ширины и формата



group: 1.0.8 ( 2014-04-09 )
(<Установка::1.0.8>)

[*] при заполнении элементов книги Excel (ячейки, строки и листы) используется
    буферизация с последующей записью в clob



group: 1.0.7 ( 2014-04-03 )
(<Установка::1.0.7>)

[*] Права на использование модуля выданы всем пользователям.



group: 1.0.6 ( 2014-01-31 )
(<Установка::1.0.6>)

[+] добавлен стиль Text для возможности записи чисел в виде строки
    (см. <pkg_ExcelCreate.Text_StyleName>)



group: 1.0.5 ( 2013-08-06 )
(<Установка::1.0.5>)

[+] добавлена возможность слияния ячеек (по горизонтали и по вертикали)

[+] добавлена возможность добавлять значение в ячейку с определенным номером,
    без необходимости заполнять все предыдущие ячейки

[+] добавлены процедуры по созданию итоговых автосумм для числовых столбцов
    (см. <pkg_ExcelCreate.addAutoSum> и <pkg_ExcelCreate.addAutoSumByName>)



group: 1.0.4 ( 2013-05-24 )
(<Установка::1.0.4>)

[+] добавлен стиль General (см. <pkg_ExcelCreate.General_StyleName>)



group: 1.0.3 ( 2013-03-04 )
(<Установка::1.0.3>)

[+] добавлена процедура очистки списка колонок в документе Excel для возможности
    формирования разных по форме отчетов на разных листах одного документа Excel
    (см. <pkg_ExcelCreate.clearColumnList>)

[*] удалены DEPRECATED процедуры / функции



group: 1.0.2 ( 2012-03-26 )
(<Установка::1.0.2>)

[!] исправлена ошибка при которой мог сформироваться некорректный xml-файл, если
    в значениях ячеек использовались служебные символы xml (<, >, ', ", &)



group: 1.0.1 ( 2012-02-13 )
(<Установка::1.0.1>)

[+] добавлена возможность создавать свои стили (путем указания соответствующих
    параметров) и использовать их при создании значений в ячейках

[+] добавлена возможность рисовать границы вокруг ячеек



group: 1.0.0 ( 2011-11-22 )
(<Установка::1.0.0>)

[+] реализована первая версия модуля по формированию Excel отчетов
