title: Версии



group: 1.2.1 ( 2016-12-27)

[!] Обеспечена успешная установка модуля в случае наличия доступного пакета
    pkg_Operator, не относящегося к модулю AccessOperator ( не содержащего
    функцию getCurrentUserId).

[!] Исправлен префикс в именах индексов на таблице <mod_install_action>.

[*] Из настроечной таблицы <mod_install_type> удалено поле operator_id
    ( для уменьшения зависимости от модуля AccessOperator).



group: 1.2.0 ( 2015-03-04)

[*] Для сохранения результата установки приложений добавлены функции
    <pkg_ModuleInfo.startAppInstall> и <pkg_ModuleInfo.finishAppInstall>
    ( вместо функции createAppInstallResult, которая стала считаться
    устаревшей, но временно сохранена для совместимости). При этом в таблицу
    <mod_app_install_result> и представление <v_mod_app_install_result>
    добавлены поля java_return_code и error_message.



group: 1.1.3 ( 2014-08-07)

[+] При выдаче прав на просмотр и сохранение информации об установках модулей
    стали выдаваться права на выборку из представлений с данными по установке
    приложений ( <v_mod_app_install_result> и <v_mod_app_install_version>).



group: 1.1.2 ( 2013-06-18)

[+] Добавлена функция <pkg_ModuleInfo.getModuleId> поиска модуля ( получения
    id).

[+] Дополнен скрипт по выдаче прав на сохранение установки (
    <Install/Grant/Last/save-install-info.sql>, выдача прав на пакет
    <pkg_ModuleInfo>); Права выдаются public.



group: 1.1.1 ( 2012-06-07)

[!] Обеспечена возможность сохранения результата установки настройки прав
    доступа в случае, если изменялись права для роли ( ранее выбрасывалось
    исключение "ORA-20195: Указан несуществующий пользователь БД").



group: 1.1.0 ( 2012-04-02)

[+] Добавлена возможность сохранения дополнительной информации об
    устанавливаемом модуле: путь в Subversion, из которого были получены файлы
    модуля ( поле svn_path), и информация о версии файлов модуля ( поле
    svn_version_info).

[!] В пути к корневому каталогу модуля ( поля svn_root, initial_svn_root)
    разрешено использование цифр.



group: 1.0.0 ( 2011-10-06)

[+] Реализовано сохранение информации о выполнявшихся установках и
    установленных версиях модулей.
