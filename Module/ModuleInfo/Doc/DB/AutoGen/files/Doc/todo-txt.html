Общие

[*] Удалить поле operator_id из настроечных таблиц модуля ( например, mod_install_type), т.к.  данный модуль ставится первым и он не должен зависеть от модуля AccessOperator.

[+] Добавить в таблицы mod_install_action, mod_install_result, mod_app_install_result поле svn_root для сохранения используемого при установке значения чтобы исключить изменение результатов выборки по ранее установленным версиям после переименования модуля ( также нужно использовать добавленное поле в соответствующих представлениях).

[+] Добавить в представление v_mod_install_object поля svn_path и svn_version_info ( возможно также полный путь к исходному файлу объекта в SVN).

[*] Более четко сохранять информацию о принадлежности исходного файла к определенной части модуля с учетом возможности загрузки одного и того же файла в составе разных частей модуля ( сейчас в таблице mod_source_file присутствует Id части модуля, что подразумевает принадлежность файла к одной части модуля).

[*] Уточнить комментарий к полю privs_user таблицы mod_install_result ( и связанных с ней представлений) в связи с возможностью указания в этом поле имени роли ( возможно также целесообразно переименовать это поле).

[+] Добавить скрипты отмены выдачи прав.
