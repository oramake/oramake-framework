Описание

Модуль, отвечающий за отслеживание работы процессов

Возможности

  • Отслеживание работы батчей и прерывание при превышении опр.  интервала ( pkg_ProcessMonitor.checkBatchExecution)
  • Трассировка сессий батчей при превышении опр.  интервала ( pkg_ProcessMonitor.checkBatchExecution)
  • Трассировка собственной сессии ( pkg_ProcessMonitor.sqlTraceOn, pkg_ProcessMonitor.sendTrace)
  • Трассировка существующей сессии ( pkg_ProcessMonitor.sqlTraceOn, pkg_ProcessMonitor.sendTrace)
