title: Версии



group: 1.3.1 ( 2018-11-01)
(<Установка::1.3.1>)

[*] Уточнен способ определения максимального времени выполнения пакетного
    задания в связи с изменениями в модуле Scheduler версии 4.6.0.



group: 1.3.0 ( 2017-12-27)
(<Установка::1.3.0>)

[+] Добавлено отслеживание работы обработчиков, т.е. батчей с расписанием
    "немедленно". Добавлены соответствующие параметры для батча
    CheckBatchExecution.



group: 1.2.0 ( 2017-12-12)
(<Установка::1.2.0>)

[!] Игнорирование всех символов в строке размера памяти, кроме цифр.



group: 1.1.1 ( 2016-10-20)
(<Установка::1.1.1>)

[!] Использование объекта opt_option_list_t модуля Option.



group: 1.1.0.2 ( 2016-07-07)
(<Установка::1.1.0.2>)

[*] Добавлено логирование памяти PGA ( Program Global Area) сеансов, а также
    суммарно занимаемой PGA.



group: 1.1.0.1 ( 2016-04-15)
(<Установка::1.1.0.1>)

[+] Добавлено логирование размера памяти, занимаемой процессом ядра Oracle (
    реализовано и проверено для OS Microsoft Windows версии не выше 7).



group: 1.1.0 ( 2014-06-30)
(<Установка::1.1.0>)

[+] Добавлено слежение за использованием памяти.



group: 1.0.1 ( 2011-11-21 )
(<Установка::1.0.1>)

[*] Дополнено описание уровней трассировки в документации.

[*] Код приведен в соответствие с новыми стандартами.



group: 1.0.0.1( 2011-03-17)

[!] Исправлена ошибка: трассировка уставливалась только для текущей
    сессии из-за ошибочного условия для использования пакета
    dbms_system при установке трассировки.



group: 1.0.0 ( 2010-06-28)

[*] Выложена первоначальная версия модуля.

[+] Реализация батча CheckBatchExecution перенесена в модуль
    ProcessMonitor.
