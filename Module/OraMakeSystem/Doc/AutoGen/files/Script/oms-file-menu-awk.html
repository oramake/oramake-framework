getComparedString

Получает строку для сортировки В начале номера встретившихся правил затем путь без расширения, где все числа, между символами “.”, преобразованы в строки, как 10 в степени 10 минус <исходное число> и дополнены нулями слева до 10 символов.  Преобразует результат в нижний регистр.

Входные параметры

filePathпуть к файлу
fileIndexиндекс файла
ruleMaxCountмаксимальное количество встретившихся правил
fileList_RuleNumberListсписки номеров встретившихся правил

Возврат

  • строка для сортировки
