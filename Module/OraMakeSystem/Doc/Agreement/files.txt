Title: Файлы

Файлы общего назначения:

readme.txt              - общее описание модуля
todo.txt                - возможные доработки
bugs.txt                - известные ошибки

Скрипты:

  Скрипт представляет собой текстовый файл в кодировке Windows-1251, который
может быть синтаксически корректно выполнен с помощью SQL*Plus.

  Исходный текст объекта БД должен находиться в скрипте, имя которого совпадает
с именем объекта, а расширение соответствует типу объекта.

  В результате выполнения скрипта программная единица ( спецификация пакета,
тело пакета, процедура и т.д.) должна пересоздаваться.

  В скрипт создания таблицы должены входить DDL-иструкции для создания самой
таблицы и связанных с ней объектов ( индексов, ограничений целостности) за
исключением триггеров. Для обеспечения корректного порядка создания объектов
ссылочные ограничения ( внешние ключи) таблицы должны находиться в отдельном
скрипте ( файл "<имя таблицы>.con").


Расширения файлов со скриптами:

*.con                   - ссылочные ограничения ( внешние ключи) таблицы
*.jav                   - Java-исходник
*.pks                   - спецификация пакета
*.pkb                   - тело пакета
*.prc                   - процедура или функция ( не входящая в пакет)
*.snp                   - материализованное представление
*.sqs                   - последовательность
*.tab                   - таблица ( включая ограничения целостности и индексы)
*.trg                   - триггер
*.typ                   - SQL-тип ( объявление)
*.tyb                   - SQL-тип ( реализация)
*.vw                    - представление
*.sql                   - остальные скрипты

Расширения файлов других типов:
*.dat                   - файл с данными для загрузки с помощью SQL*Loader
*.ctl                   - контрольный файл для SQL*Loader
*.jar                   - Java-библиотека

Файлы для загрузки пакетных заданий ( батчей модуля Scheduler):
*.job.sql               - файл данных задания ( job)
batch.xml               - файл данных пакетного задания
batch_config.xml        - файл настроек пакетного задания ( расписание, опции и т.п.)
