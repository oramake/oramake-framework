title: Установка

Реализуется с помощью целей сборки модуля из make-целей <Установка модуля в БД>.
Для корректной установки должна быть установлена актуальная версия OMS
( см. <Версия OMS-файлов>).

Параметры установки:
INSTALL_VERSION               - устанавливаемая версия модуля ( Last при
                                первоначальной установке или номер при
                                обновлении)
LOAD_OPERATORID               - логин и пароль оператора, выполняющего установку
                                ( формат: login/password)
LOAD_USERID                   - основная схема для установки модуля
                                ( формат: username/password@dbName)
LOAD_USERID2                  - пользователь для установки функционала TextService в БД
                                ( формат: username/password@dbName)



group: 1.3.0
(<Версии::1.3.0 ( 2017-02-14)>)

Порядок установки:

- установка в основную схему;

(code)

make install INSTALL_VERSION=1.3.0 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

(end code)

- выдача прав пользователям;

(code)

make grant LOAD_USERID=???/???@??? TO_USERNAME=??? LOAD_OPERATORID=???/???

(end code)



group: Первоначальная


- установка модуля в основную схему:

(code)

make install INSTALL_VERSION=Last LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

(end code)


- установка пакета <pkg_TextService> в схему ctxsys:

(code)
-- Установка пакетов модуля в схему CTXSYS
make load LOAD_USERID2=ctxsys/<password>@dbName

-- Выдача права пользователю CTXSYS на создание синонима в произвольной пользовательской схеме
make grant LOAD_USERID2=sys/<password>@dbName TO_USERNAME=CTXSYS GRANT_SCRIPT=sys-privs

-- Выдача прав на объекты модуля произвольному пользователю и создание синонима
-- в пользовательской схеме
make grant LOAD_USERID2=ctxsys/<password>@dbName TO_USERNAME=<userSchema> GRANT_SCRIPT=ctxsys
(end)
