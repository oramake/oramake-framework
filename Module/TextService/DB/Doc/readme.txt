title: Описание

Модуль предоставляет сервисные и утилитарные методы работы с функционалом поиска по текстовым данным.

Сервисный функционал предназначен для "проксирования" методов Oracle Text, которые могут быть вызваны только под CTXSYS. Сервисный функционал устанавливается в схему CTXSYS и 
вызывается из прикладных схем.
