title: Версии



group: 1.3.0 ( 2015-09-10)

[+] Добавлен скрипт для выдачи прав для использования модуля.



group: 1.2.2 ( 2013-07-03)

[*] Функции проверки корректности ИНН стали считать допустимыми корректные
    10-значные ИНН юридических лиц.



group: 1.2.1 ( 2012-12-21)

[+] Добавлена функция <pkg_FormatData.checkVin> ( <pkg_FormatData.checkVinExpr>)
    для проверки корректности VIN ( идентификационного номера автомобиля) ( по
    длине и использованию только допустимых символов).

[+] Добавлена функция <pkg_FormatData.checkPts> ( <pkg_FormatData.checkPtsExpr>)
    для проверки корректности серии и номера ПТС ( паспорта транспортного
    средства) ( по шаблону).



group: 1.2.0 ( 2011-12-29)

[+] Добавлены функции проверки корректности номера заграничного паспорта
    и водительского удостоверения ( по шаблону).



group: 1.1.0 ( 2011-03-24)

[+] Добавлены функции проверки корректности ИНН и номера пенсионного
    свидетельства с помощью проверки контрольных сумм.

[*] Уточнено оформление модуля в соответствии с текущими соглашениями.



group: 1.0.5 ( 2010-07-27)

[!] Исправлена ошибка в скрипте первоначальной установки последней версии
    модуля.
