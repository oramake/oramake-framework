Test/expr-function.sql

Тест функций, возвращающих выражения для динамического SQL ( функции *Expr пакета pkg_FormatData).  Функция вызывается динамически и выводится выходное значение, полученное на основе входного значения.

Параметры

funcNameимя функции ( например, “getBaseFirstNameExpr”)
inDataвходное значение для поля
addonParamListсписок значений для дополнительных параметров функции ( например, “, 1” передает 1 в качестве второго параметра)
