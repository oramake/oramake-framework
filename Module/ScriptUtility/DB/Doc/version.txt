title: Версии



group: 1.2.1 ( 2014-08-20)

[+] Добавлена процедура генерации скриптов создания временных таблиц для
    обновления интерфейсных таблиц по представлениям с исходными данными
    ( <pkg_ScriptUtility.generateInterfaceTempTable>).



group: 1.2.0 ( 2013-10-25)

[+] Добавлена процедура генерации скриптов создания интерфейсных таблиц по
    представлениям с исходными данными
    ( <pkg_ScriptUtility.generateInterfaceTable>).



group: 1.1.1 ( 2013-08-01)

[*] Изменён текст генерируемых функция для Api в соответствии со стандартами.

[*] Колонка типа дата в функции поиска соответствует двум параметрам границ
    интервала.



group: 1.1.0 ( 2011-11-14)

[+] Добавлена процедура генерации sql вставки fake-значений в таблицу.



group: 1.0.1 ( 2011-09-29)

[*] При добавлении записи в базовую таблицу с исторической структурой для поля
    operator_id по умолчанию стало использоваться значение change_operator_id,
    а затем Id текущего оператора ( ранее только Id текущего оператора).

[*] Убраны значения по умолчанию для полей deleted, change_number, change_date
    в скрипте создания исторической таблицы.

[*] Решена проблема с возникновением ошибки вида
    'SP2-0734: unknown command beginning "..." - rest of line ignored.'
    при загрузке тела пакета <pkg_ScriptUtility> с помощью SQL*Plus.



group: 1.0.0

[+] Реализована генерация скриптов исторической структуры и другие
    вспомогательные функции.
