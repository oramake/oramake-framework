[*] При добавлении записи в базовую таблицу с исторической структурой для поля operator_id по умолчанию стало использоваться значение change_operator_id, а затем Id текущего оператора ( ранее только Id текущего оператора).

[*] Убраны значения по умолчанию для полей deleted, change_number, change_date в скрипте создания исторической таблицы.

[*] Решена проблема с возникновением ошибки вида ‘SP2-0734: unknown command beginning “...”  - rest of line ignored.’  при загрузке тела пакета pkg_ScriptUtility с помощью SQL*Plus.

1.0.0

[+] Реализована генерация скриптов исторической структуры и другие вспомогательные функции.
