1.4.0

(Версии::1.4.0 ( 2019-08-16))

Версия устанавливается в любую целевую БД.

Порядок установки

  • установка в основную схему ( устанавливается под основным пользователем, который должен иметь права на выполнение опции alter system)
make install INSTALL_VERSION=1.4.0 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

1.3.2

(Версии::1.3.2 ( 2018-05-04))

Версия устанавливается в любую целевую БД.

Порядок установки

  • Отключение job, необходимое для корректной установки модуля, устанавливается под основным пользователем, в схему которого установлен модуль TextCreate и пользователь имеет права на выполнение опции alter system
make install-before INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???
  • установка в основную схему
make install INSTALL_VERSION=1.3.2 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???
  • Включение job, отключенных командой выше, для корректной установки модуля
make install-after INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

make install INSTALL_VERSION=Last LOAD_USERID=login/password@dbName LOAD_OPERATORID=operatorLogin/operatorPassword

make grant LOAD_USERID=login/password@dbName GRANT_SCRIPT=all-to-public TO_USERNAME=public LOAD_OPERATORID=operatorLogin/operatorPassword
