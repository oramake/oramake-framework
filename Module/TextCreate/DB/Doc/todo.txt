Title: Доработки



[*] При необходимости использования в других модуля сжатия данных рекомендуется вынести java-библиотеку в отдельный
модуль.

[!] Данный метод сжатия zip реализован на базе стандарных оракловых build-in java классов, в которых не курректно поддерживается
киррилические символы в названиях файлов. При необходимости можно провести доработку, позволяющую исправить эту ситуацию за счёт
использования библиотек Apache Ant (http://ant.apache.org/bindownload.cgi).
