title: Установка



group: Общие замечания

Параметры установки:
INSTALL_VERSION               - устанавливаемая версия модуля ( Last при
                                первоначальной установке или номер при
                                обновлении)
LOAD_OPERATORID               - логин и пароль оператора, выполняющего установку
                                ( формат: login/password)
LOAD_USERID                   - основной пользователь для установки в БД
                                ( формат: username/password@dbName)
LOAD_USERID2                  - Пользователь для запуска/отключения батчей
                                ( формат: username/password@dbName)




group: 1.4.0
(<Версии::1.4.0 ( 2019-08-16)>)

Версия устанавливается в любую целевую БД.

Порядок установки:

- установка в основную схему ( устанавливается под основным пользователем,
  который должен иметь права на выполнение опции *alter system*)
> make install INSTALL_VERSION=1.4.0 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???



group: 1.3.2
(<Версии::1.3.2 ( 2018-05-04)>)

Версия устанавливается в любую целевую БД.

Порядок установки:

- Отключение job, необходимое для корректной установки модуля, устанавливается под основным пользователем,
  в схему которого установлен модуль *TextCreate* и пользователь имеет права на выполнение опции *alter system*
> make install-before INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

- установка в основную схему
> make install INSTALL_VERSION=1.3.2 LOAD_USERID=???/???@??? LOAD_OPERATORID=???/???

- Включение job, отключенных командой выше, для корректной установки модуля
> make install-after INSTALL_VERSION=Last LOAD_USERID2=???/???@??? LOAD_OPERATORID=???/???

group: 1.1.0

Последовательность установки:

- выдача прав на объекты модуля всем пользователям и создание PUBLIC-синонима
> make grant LOAD_USERID=login/password@dbName GRANT_SCRIPT=all-to-public


group: Первоначальная установка

Последовательность установки:

- проливка в основную схему
> make install INSTALL_VERSION=Last LOAD_USERID=login/password@dbName LOAD_OPERATORID=operatorLogin/operatorPassword

- выдача прав на объекты модуля всем пользователям и создание PUBLIC-синонима;
> make grant LOAD_USERID=login/password@dbName GRANT_SCRIPT=all-to-public TO_USERNAME=public LOAD_OPERATORID=operatorLogin/operatorPassword
