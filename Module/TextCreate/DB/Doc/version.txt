title: Версии



group: 1.4.0 ( 2019-08-16)
(<Установка::1.4.0>)

[+] Добавлены функции <pkg_TextCreate.base64Decode>,
    <pkg_TextCreate.base64Encode> декодирования Base64 в BLOB ( двоичных данных в
    текстовые в кодировке Base64 и обратно).



group: 1.3.2 ( 2018-05-04)
(<Установка::1.3.2>)

[*] Изменен пакет <pkg_TextCreate>. Добавлены константы

- <pkg_TextCreate.UTF8_CharsetName>
- <pkg_TextCreate.Windows1251_CharsetName>

[*] Изменен пакет <pkg_TextCreate>. Добавлен параметр charsetName для указания кодировки в следующих функциях

- <pkg_TextCreate.getZip>
- <pkg_TextCreate.convertToClob>
- <pkg_TextCreate.convertToBlob>


group: 1.3.1 ( 2015-07-07)
(<Установка::1.3.1>)

[+] В процедуру <pkg_TextCreate.newText> добавлена принудительная
    очистка временного lob'а, т.к. Oracle его не очищает и при
    повторном вызове освобождение занимаемого lob'ом места
    не происходит, вместо этого открывается новый временный lob.


group: 1.3.0 ( 2015-02-13)
(<Установка::1.3.0>)

[+] Добавлены функции <pkg_TextCreate.convertToBlob>,
    <pkg_TextCreate.convertToClob> конвертации BLOB в CLOB ( двоичных данных в
    текстовые и обратно).


group: 1.2.2 ( 2014-04-09)
(<Установка::1.2.2>)

[+] Добавлена перегруженная функция <pkg_TextCreate.append ( clob )> для
    добавления текстовой информации в виде clob

[!] Исправлена ошибка number overflow при выгрузке в zip-архиве файла, размер
    которого превышает 2Гб.


group: 1.2.1 ( 2013-12-20)
(<Установка::1.2.1>)

[*] Обеспечено корректное отображение имени файла с кириллицей в zip-архиве,
    создаваемом функцией <pkg_TextCreate.getZip>.


group: 1.2.0
(<Установка::1.2.0>)

В интерфейсный пакет добавлана функция getZip, позволяющая сохранять сжатые архивы типа zip.
