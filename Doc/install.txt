Установка
====================

Последовательность первоначальной установки модулей в БД (команды выполняются в корневом
каталоге проекта):

- создание вспомогательного пользователя с расширенными привилегиями
  (om_admin с паролем "om_admin")

SQL> @Module/UserCreate/create-user-admin.sql om_admin

  (выполняется в SQL*Plus под пользователем, имеющим достаточно привилегий для
  успешного выполнения скрипта;  при запросе имени табличного пространства
  указывается USERS)

- создание основного пользователя для установки модулей
  (om_main с паролем "om_main")

SQL> @Module/UserCreate/create-user-main.sql om_main

  (выполняется в SQL*Plus под пользователем om_admin, созданным на предыдущем
  шаге установки, либо под любым другим пользователем, имеющим достаточно
  привилегий для успешного выполнения скрипта; при запросе имени табличного
  пространства указывается USERS)

- установка OraMakeSystem

Для устновки модулей в БД нужно установить OraMakeSystem из
Module/OraMakeSystem согласно документации:

Module/OraMakeSystem/Doc/Html/ru/index.html
(plain text: Module/OraMakeSystem/Doc/install.txt)

- установка модулей в БД

> make install MAIN_USERID=om_main/om_main@orcl ADMIN_USERID=om_admin/om_admin@orcl SQL_DEFINE=indexTablespace=USERS

  (для индексов используется явно заданное табличное пространство USERS,
  описание дополнительных параметров установки есть в комментариях
  в файле Makefile корневого каталога проекта)


В результате будут установлены основные модули (перечисленные в настроечном
параметре allModuleList файла Makefile корневого каталога проекта).
Остальные модули устанавливаются индивидуально согласно документации по
их установке (см. Module/<moduleName>/Doc/DB/AutoGen/index.html, раздел
"O модуле -> Установка").
